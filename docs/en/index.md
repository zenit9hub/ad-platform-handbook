## About This Site

This site is designed to improve practical understanding of ad platform architecture without exposing customer-specific or internal operational details.

## Recommended Reading Path

1. Start with the fundamentals.
2. Move into standards such as OpenRTB and VAST.
3. Continue into trust, verification, reconciliation, and Web3-oriented extension points.

## Navigation Model

- Once you enter the documentation, the sidebar keeps all categories visible.
- You can move across categories without returning to the homepage.
- The GitHub repository remains available through the header icon, while homepage actions focus on learning paths.

## Scope

- Public, generalized knowledge only
- No customer-specific implementation details
- No production endpoints, credentials, or internal logs
