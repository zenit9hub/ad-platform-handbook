# Trust · Web3 Lab

## Purpose

This category is a later-stage lab for readers who already understand the core standards, measurement, verification, and implementation structure of ad platforms. It does not present Web3 as a baseline requirement. Instead, it examines where SSI, blockchain, cryptographic proof, and provenance may become useful extension points.
