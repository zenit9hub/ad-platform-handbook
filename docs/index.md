## 이 사이트에 대해

이 사이트는 고객사 특화 정보나 내부 운영 정보를 드러내지 않으면서도, 광고플랫폼 구조와 표준을 실무적으로 이해할 수 있도록 설계되었습니다.

## 추천 학습 경로

1. 먼저 기초 개념부터 읽습니다.
2. 이후 OpenRTB, VAST 같은 표준으로 이동합니다.
3. 다음으로 측정, 신뢰, 구현 패턴을 확장해서 읽습니다.

## 다루는 범위

- 공개 가능한 일반화된 지식만 다룹니다.
- 고객사 특화 구현 내용은 포함하지 않습니다.
- 실제 운영 endpoint, credential, 내부 로그는 다루지 않습니다.
