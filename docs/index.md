## 이 사이트에 대해

이 사이트는 고객사 특화 정보나 내부 운영 정보를 드러내지 않으면서도, 광고플랫폼 구조와 표준을 실무적으로 이해할 수 있도록 설계되었습니다.

## 추천 학습 경로

1. 먼저 기초 개념부터 읽습니다.
2. 이후 OpenRTB와 표준 객체 구조를 읽습니다.
3. 다음으로 adm, 렌더링, runtime 실행 주체를 이해합니다.
4. 이어서 측정, 검증, 정합성, 공급 경로 해석을 읽습니다.
5. 그 다음 채널별 차이와 구현 패턴을 읽습니다.
6. 마지막으로 Trust · Web3 실험실과 레퍼런스 문서로 확장합니다.

## 탐색 방식

- 한 번 문서에 진입하면 사이드바에서 모든 카테고리와 문서를 계속 탐색할 수 있습니다.
- 현재 읽는 카테고리뿐 아니라 다른 카테고리의 문서도 바로 이동할 수 있도록 구성합니다.
- GitHub 저장소 링크는 상단 아이콘으로 유지하고, 홈에서는 학습 경로 중심 CTA만 노출합니다.

## 다루는 범위

- 공개 가능한 일반화된 지식만 다룹니다.
- 고객사 특화 구현 내용은 포함하지 않습니다.
- 실제 운영 endpoint, credential, 내부 로그는 다루지 않습니다.
